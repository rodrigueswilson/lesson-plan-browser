# Scripts Directory

This directory contains utility scripts for the Bilingual Lesson Plan Builder.

## Organization

- **`diagnostics/`** - Diagnostic and debugging scripts for specific issues
- **Root scripts** - General utility scripts for maintenance and operations

## Main Vocabulary/Frames Fix Scripts

If you need to fix vocabulary/frames issues:

1. **Fix all plans**: `python fix_all_plans.py` (in root directory)
2. **Fix specific plan**: `python update_plan_supabase.py` (in root directory)
3. **Diagnose issues**: See `diagnostics/` folder scripts
