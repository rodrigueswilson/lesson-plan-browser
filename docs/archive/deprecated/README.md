# Deprecated Files Archive

This directory contains archived deprecated files that were previously located in the `deprecated/` directory at the project root.

## Organization

Files are organized into three categories:

- **documentation/**: Deprecated documentation files (prompts, READMEs)
- **strategies/**: Deprecated bilingual strategy JSON files (old versions)
- **samples/**: Deprecated sample lesson plan files

## Archive Index

See [ARCHIVE_INDEX.md](ARCHIVE_INDEX.md) for a complete list of all archived files with their original paths, archive locations, dates, and reasons for archiving.

## Current Status

All files from the `deprecated/` directory have been moved to this archive. The original `deprecated/` directory now contains only a README pointing to this archive.
