# Bilingual Teaching Strategies Dictionary

This file is a **human-readable companion** to the JSON dictionary used in an MCP environment for transforming primary teachers’ lesson plans into bilingual lesson plans.
It explains the dictionary structure and describes 27 bilingual teaching strategies in detail.

## 1. Explanation of the Dictionary and How to Use It

Each strategy entry contains the following fields:

- **Strategy ID**: Unique identifier (used in JSON lookups).
- **Name**: Common label for the strategy.
- **Aliases**: Alternative names used in research or practice.
- **Evidence Tier**: Level of research support (research-backed, practice-based, or mixed).
- **High-Level Categories**: One or more of:
  - Language Skill Development
  - Instructional Frameworks & Models
  - Cross-Linguistic & Biliteracy
  - Assessment/Scaffolding/Regulation
- **Primary Skill**: Speaking, Listening, Reading, Writing, or Integrated.
- **Secondary Skills**: Other supported skills.
- **Skill Weights**: Approximate emphasis on each skill.
- **Delivery Mode**: Push-in, whole-group, or small-group.
- **L1 Mode**: How/if the first language is used (Translanguaging, Preview-Review, Strategic Code-Switch, or None).
- **Core Principle**: Concise rationale.
- **Applicable Contexts**: Classroom situations where this strategy fits.
- **Implementation Steps**: Practical how-to guidance.
- **Primary Benefits**: Expected outcomes.
- **Research Foundation**: Key scholars or references.
- **Cross-References**: Related strategies to combine.

### How to Use This File

1. Start from the **content objective** in the primary teacher’s lesson plan.
2. Identify the **target language skill(s)** (speaking, listening, reading, writing).
3. Look up relevant strategies that support those skills.
4. Use the **implementation steps** and **benefits** to guide the creation of a **bilingual language objective**.
5. Combine with the **objective templates and phrasebanks** in the JSON file to generate ready-to-use bilingual objectives.

### Strategy ID: bilingual_dictionary

**Name:** Bilingual Dictionary and Glossary Use  

**Aliases:** Bilingual glossaries  

**Evidence Tier:** practice-based/mixed  

**High-Level Categories:** Language Skill Development, Cross-Linguistic & Biliteracy  

**Primary Skill:** reading  

**Secondary Skills:** writing  

**Skill Weights:** speaking: 0.0; listening: 0.0; reading: 0.5; writing: 0.5  

**Delivery Mode:** push-in, small-group, whole-group  

**L1 Mode:** translanguaging, preview-review  


**Core Principle:**  
Leverage L1–L2 dictionaries to clarify new terms and support independent comprehension.  


**Applicable Contexts:**  
- Self-directed reading or research  
- Content lessons with dense academic vocabulary  
- Writing tasks requiring precise word choice  


**Implementation Steps:**  
1. Model context-first inference, then confirm with a bilingual dictionary.  
1. Embed routine look-ups in reading/writing; teach efficient dictionary skills.  
1. Verify fit-in-context (e.g., reverse-translation or image check).  


**Primary Benefits:**  
- Improves comprehension of academic texts  
- Builds bilingual vocabulary and learner autonomy  


**Research Foundation:** Jim Cummins, Ellen Bialystok  

**Cross-References:** Explicit Vocabulary Instruction, Cognate Awareness
